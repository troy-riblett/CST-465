17. What is the purpose of the DOCTYPE declaration at the top of a web page

The purpose of the doctype declaration is to inform the browser of which type of html is going to be used for the document

18. What is the difference between inline and block HTML elements

Block type HTML elements expand naturally to fill their line/container and can have padding and margins. Inline elements are displayed inline with the rest of the text and thus ignore width and height type properties in addition to not clearing floats to drop to the next line automatically

19. Explain why the alt attr ibute of the img tag is important

The alt attribute is what is read in place of the image for visually impaired usability. It also is used by search engines to help determine relevance

20. What does it mean for content to be 'Above the Fold'

Content that is above the fold is content that the user does not have to scroll down to see. If content is above the fold the user is much more likely to end up seeing/noticing it

21. I often drone on about HTML being a semantic language. What do I mean, and how does this affect the way HTML should be written

HTML is a semantic language because it determines how the document is structured and not how it is to be displayed. This means that HTML should be written to structure a document and not to make it display well (which is the job of CSS)

22. A business object can be used as the model as part of the MVC pattern. Why would we create a model separate from this for use with the view?

We usually create a separate model than our business object so that we can decorate the model with constraints so that the business object is not cluttered with web-specific type code
